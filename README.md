# Glaucoma Detection Using Fundus Images

This project focuses on **automatic glaucoma detection** using retinal fundus images with the help of **image processing and deep learning techniques**.  
The goal is to assist early diagnosis of glaucoma, which can help prevent permanent vision loss.
